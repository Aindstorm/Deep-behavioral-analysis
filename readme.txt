This repo includes behavioral analysis of mfc customers based on deep models combined with random forest and gradient boosting. 

Notebooks order:
1) Data preparation
2) Data preparation prt2
3) Data preparation prt3
4) TF model (1 lag)
5) TF model (2 lags)
6) get-embeddings
7) Cluster analysis
8) Prediction (2 TF models)
9) xgboost
10) random_forest + final output

Data is under NDA, so unfortunately can't share it.
